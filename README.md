# Piloto – Agente Semiautomático

## Descripción
Piloto es un agente semiautomático diseñado para analizar código, proponer mejoras y ejecutar workflows de manera controlada, trabajando junto con la asistente “Gema”. Permite centralizar el desarrollo, pruebas y refactorización, integrándose con GitHub y enviando notificaciones de progreso vía Telegram.

## Ramas en Git

- `main` – Rama principal del proyecto. Contiene la versión estable.  
- `dev` – Desarrollo general y pruebas de integración.  
- `feature/analysis` – Funcionalidades de análisis de código.  
- `feature/refactor` – Funcionalidades de refactorización.  
- `feature/utils` – Scripts y utilidades auxiliares.  
- `feature/workflow` – Funciones de workflow semiautomático.  
- `feature/tests` – Pruebas unitarias y de integración.

## Objetivos de la Versión 1.0.0

1. Crear la estructura básica del proyecto y subirla a GitHub, incluyendo carpetas iniciales: `analysis`, `refactor`, `core`, `utils`, `tests`, `workflow`.  
2. Implementar primer workflow semiautomático: análisis de código y propuesta de cambios.  
3. Sistema de notificaciones simple: avisos de tareas completadas vía Telegram.  
4. Soporte inicial para refactorización con IA externa, con control humano para aplicar cambios.  
5. Documentación inicial y guía de uso básico.
