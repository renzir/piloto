# Piloto v1.0.1

## Descripción
Piloto es un sistema semiautomático para analizar, refactorizar y testear proyectos de código.

## Objetivos v1.0.1
1. Analizar código de proyectos existentes.
2. Aplicar refactorizaciones basadas en reglas definidas.
3. Gestionar workflow semiautomático: propuesta → aprobación → aplicación → test → notificación.
4. Almacenar historial de cambios y sesiones de análisis.

## Estructura
- core/
- analysis/
- refactor/
- utils/
- tests/
- data/
